รายการจองวันนิเทศทั้งหมด

คลิกที่ชื่อครู หรือปุ่ม "แนบแผนการสอน" เพื่อแนบไฟล์ PDF แผนการจัดการเรียนรู้ขึ้น Google Drive ย้อนหลัง

วันที่รับนิเทศ คาบเรียน / เวลา ครูผู้รับการนิเทศ (คลิกเพื่ออัปโหลดไฟล์) รายวิชา ระดับชั้น ห้องเรียน แผนการสอนบน Drive การจัดการ

ยังไม่มีประวัติการจอง

กรุณากลับไปที่หน้าทำรายการจองเพื่อเริ่มจองคิวแรก

ตั้งค่าระบบ Google Apps Script

กรอก URL เว็บแอปที่ได้รับจากการ Deploy สิทธิ์เป็นแบบ "Anyone" เพื่อเชื่อมต่อระบบส่งไฟล์และจองข้อมูล

ขั้นตอนการสร้างระบบบันทึกและอัปโหลดเข้า Google Drive ไร้การเข้าสู่ระบบ

  1. สร้าง Google Sheet ใหม่ และตั้งชื่อแผ่นงานแรกสุดว่า Teachers และแผ่นงานที่สองว่า Bookings
  2. ในชีต Teachers ช่อง A1 ใส่หัวข้อว่า Teacher Name จากนั้นกรอกรายชื่อครูในแถวถัดๆ ไป (A2, A3, ...)
  3. ใน Google Sheet เปิดเมนู Extensions (ส่วนขยาย) > Apps Script
  4. คัดลอกโค้ดสคริปต์ "โค้ดระบบหลังบ้าน" (ด้านล่างนี้) ไปวางในแท็บแก้ไขไฟล์ Code.gs
  5. สร้างโฟลเดอร์สำหรับรับไฟล์แผนการสอน PDF ใน Google Drive ของคุณ และนำ Folder ID ของคุณ มาใส่ในตัวแปร DRIVE_FOLDER_ID ภายในโค้ดสคริปต์ (ห้ามใช้ตัวอย่าง ID เดิมเด็ดขาด)
  6. กด Save และกดปุ่ม Deploy > New deployment (การติดตั้งใช้งานใหม่)
  7. เลือกประเภทเป็น Web App, รันภายใต้บัญชี: "Me" (ตัวท่านเอง) และตั้งค่าผู้เข้าถึงเป็น "Everyone" (ทุกคน) เพื่อให้ครูอัปโหลดได้แบบไม่ต้อง login
  8. คัดลอก Web App URL นำมาวางในช่องด้านบนแล้วกด "บันทึกการเชื่อมต่อ"
